# Hive

A general-purpose framework for building autonomous agent organizations.

> *"In an agent company, the signals of scaling demand are **backlog** and **focus**."*
>
> Backlog is straightforward — it exists in modern organizations.
> Focus is agent-specific: an idle agent costs nothing, but an agent juggling ten different things in its context and memory drops the quality of its work. You don't scale to save cost. You scale to preserve focus.
