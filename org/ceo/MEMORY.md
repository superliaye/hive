## Lessons Learned
- Agents default to "yes" — shared skills (scope-guard, escalation, status-protocol, skeptic-review) are critical guardrails.
- Full-context handoffs waste tokens — agents should ask clarification questions instead.
- CEO-specific skills (delegation, prioritization, board-protocol) prevent the CEO from doing subordinates' work.
